## Getting Started

Run the CLI:

```console
# Vanilla React Native
npx react-native init AwesomeProject
npx ignite-cli new AwesomeApp

# Expo-powered
expo init
npx create-react-native-app
npx ignite-cli new AwesomeApp --expo
```
